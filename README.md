# 📋 Smart Task Manager (MERN Stack)

A modern Full-Stack Task Management dashboard built using the MERN architecture. It features an automated backend notification system, real-time visual data analytics, interactive UI highlights, and smart task scheduling.

---

## 🚀 Key Features

* **Dynamic Task Board:** Create, filter, track, and manage operational deadlines.
* **Visual Analytics Engine:** Fully interactive conic-gradient donut chart calculating **Completed**, **Pending**, and **Missed** metrics instantly.
* **Timeframe Filters:** Easily toggle task dashboards between **Monthly** and **Yearly** performance scopes.
* **Quick Insights Panel:** Real-time percentage panels for tracking on-time performance trends and overdue rates.
* **Smart Layout UI Rules:** Active state tracking across global navigational components via custom active router linkages.
* **Automated Background Cron System:** Integrated `node-cron` task monitoring that scans records hourly to flag missed deadlines and fire off email notifications.

---

## 🛠️ Tech Stack

* **Frontend:** React.js, React Router DOM v6, CSS3 Custom Variables, HTML5
* **Backend:** Node.js, Express.js
* **Database:** MongoDB, Mongoose ODM
* **Utilities:** Node-Cron (Scheduler), Axios (API client), Nodemailer (Email Alerts)
